# NewsPulse – AI Trend & Sentiment Analysis

NewsPulse is an AI-based news analytics system that detects trending topics and analyzes sentiment using NLP and machine learning.

## Features
- Trend Detection
- Sentiment Analysis
- Flask Dashboard
- Login Authentication
- Data Visualization

## Technologies
Python, spaCy, TF-IDF, Logistic Regression, Flask
